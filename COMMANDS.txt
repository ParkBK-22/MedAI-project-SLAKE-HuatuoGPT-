## [1단계] 프로젝트 클론 (처음 한 번만)
git clone https://github.com/YourUserName/MedAI-project-SLAKE-HuatuoGPT.git
cd MedAI-project-SLAKE-HuatuoGPT


## [2단계] 가상환경 설정

### Linux/Mac:
python -m venv venv
source venv/bin/activate

### Windows PowerShell:
python -m venv venv
.\venv\Scripts\Activate.ps1

### Windows CMD:
python -m venv venv
venv\Scripts\activate.bat


## [3단계] 기본 패키지 업그레이드
pip install --upgrade pip setuptools wheel


## [4단계] PyTorch + CUDA 설치 (RTX 4090 기준)
pip install torch==2.4.0 torchvision==0.19.0 --index-url https://download.pytorch.org/whl/cu121


## [5단계] 최신 Transformers 설치
pip install git+https://github.com/huggingface/transformers.git --upgrade


## [6단계] HuatuoGPT 의존성 설치
pip install -r requirements_huatuogpt.txt


## [7단계] 환경 검증
python -c "import torch; print(f'✓ PyTorch: {torch.__version__}'); print(f'✓ CUDA: {torch.cuda.is_available()}'); print(f'✓ GPU: {torch.cuda.get_device_name() if torch.cuda.is_available() else \"N/A\"}')"


## [8단계] **HuggingFace 토큰 설정** (필수!)

### 방법 1: 환경변수 설정 (권장)

#### Linux/Mac:
export HF_TOKEN='your_huggingface_token_here'

#### Windows PowerShell:
$env:HF_TOKEN='your_huggingface_token_here'

#### Windows CMD:
set HF_TOKEN=your_huggingface_token_here

### 방법 2: HuggingFace CLI 로그인
huggingface-cli login
# 토큰 입력: https://huggingface.co/settings/tokens에서 생성

### 방법 3: 자동 설정 스크립트

python << 'EOF'
import os
from huggingface_hub import login

# 토큰 설정
token = input("Enter your HuggingFace token: ")
os.environ['HF_TOKEN'] = token

# 저장
login(token=token)
print("✓ Token saved!")
EOF

    print("📈 DIAGNOSTIC METRICS")
    print("=" * 80)
    print(f"VRS (Vision Relevance Score): {diag.get('VRS', 'N/A')}")
    print(f"L-Drop (LPF Performance Drop): {diag.get('L_drop', 'N/A')}")
    print(f"K-Ratio (HPF vs LPF): {diag.get('K_ratio', 'N/A')}")
EOF

================================================================================
📊 결과 해석
================================================================================

### VRS (Vision Relevance Score): 
- 0.0: 시각 정보 거의 무관
- 0.1-0.3: 약간 중요
- > 0.3: 매우 중요 ✅

### L-Drop (LPF에 의한 성능 저하):
- > 0.1: LPF가 중요한 정보 포함

### K-Ratio (고주파 vs 저주파):
- > 1: 고주파(디테일) 중요
- < 1: 저주파(구조) 중요

### Question Type별 성능:
- Location/Shape: 시각 정보 필수
- Organ/Modality: 의료 지식 중심
